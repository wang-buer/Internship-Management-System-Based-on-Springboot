# 基于springboot的实习管理系统



## 1、项目介绍

基于springboot和vue的实习管理系统拥有四种角色：管理员、教师、学生和实习单位，功能如下：

![架构](https://www.codeshop.fun/Typora-Images/202402171756898.jpg)

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：html、VUE

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
